                    • fileMode

                      • Similar to a UNIX permission, sets the file mode of the files included.

                        THIS IS AN OCTAL VALUE. Format: (User)(Group)(Other) where each component is a sum of Read = 4, Write = 2, and Execute = 1.

                        • For example, the value 0644 translates to User read-write, Group and Other read-only. The default value is 0644 (more on unix-style permissions)  

                    • directoryMode

                      • Similar to a UNIX permission, sets the directory mode of the directories included.

                        THIS IS AN OCTAL VALUE. Format: (User)(Group)(Other) where each component is a sum of Read = 4, Write = 2, and Execute = 1.

                        • For example, the value 0755 translates to User read-write, Group and Other read-only. The default value is 0755. (more on unix-style permissions)
